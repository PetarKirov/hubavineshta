A highly optimized category list/tree widget for WordPress 

== Description ==

A highly optimized category list/tree widget for WordPress. Works very nice on HUGE wordpress databases, and doesn't require hundreds of MB of RAM. 

== Installation ==

1. Upload `fastcat.php` and the `fastcat-tree` folder to the `/wp-content/plugins/` directory
1. Activate the Fast Cat plugin through the 'Plugins' menu in WordPress admin
1. Drag the FastCat widget onto one of your template's sidebars, and configure it in the 'Widgets' menu in WordPress admin
